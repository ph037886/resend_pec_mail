# resend_pec_mail
重寄醫事人員教補系統測試信

本程式主要是重寄台灣醫事人員教補系統問卷

相信各職類的教學負責人應該都會有這個困擾，每年10月要處理教補系統指標，老師學生換來換去，也不知道他們提供的mail是不是正確的，這個程式就是利用爬蟲的方式，快速寄出測試信或是重寄帳號密碼信件

教補系統的網站
https://pec.mohw.gov.tw/MONITOR_INDICATOR/Monitor_Questionnaire.aspx
